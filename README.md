# Reverse-IP-Tools

### Instalasi
----
Membutuhkan [Node.js](https://nodejs.org/) minimal ES7.
Install dan memulai server :

```sh
$ git clone https://github.com/BlackhatCode/Reverse-IP-Tools.git
$ cd Reverse-IP-Tools
$ npm install
$ node index
```
